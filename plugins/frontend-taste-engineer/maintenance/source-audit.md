# Source audit status

Initial registry established on 2026-07-10. The weekly live monitor resolves immutable GitHub heads and license/deprecation metadata, fingerprints bounded visible text for other public sources, compares the prior run, and reports changed reachability or redirects. Separate deterministic checks cover record age, local links, provenance, secrets, and license evidence. Reports contain public metadata and hashes only; they are evidence for review and never update stable guidance.

The current release documents inaccessible and unresolved supplied sources separately. Recheck those items before treating their contents or licenses as known.
