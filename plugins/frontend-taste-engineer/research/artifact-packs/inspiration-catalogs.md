# Inspiration catalogs

Awwwards, Godly, Land-book, Lapa Ninja, Landingfolio, Mobbin, Page Flows, Screenlane, Refero, SaaS Landing Page, One Page Love, and related galleries are `inspiration-only` unless a separately reviewed item provides explicit reusable code or assets under clear terms.

## Safe extraction workflow

1. Begin with a product question: hierarchy, flow, content sequencing, responsive transformation, interaction feedback, or visual character.
2. Compare several unrelated examples instead of selecting one page to imitate.
3. Record generalized observations such as “proof appears adjacent to the commitment point” or “mobile moves comparison details into progressive disclosure.”
4. Separate common pattern from distinctive expression. Do not reproduce exact composition, text, images, logos, tokens, motion timing, illustrations, or brand styling.
5. Rebuild from the product’s own content, design thesis, constraints, semantics, and states.
6. Verify originality with a side-by-side anti-copy review and remove any unmistakably source-specific expression.

## What not to infer

Awards, likes, curation, popularity, screenshots, and polished marketing do not prove accessibility, performance, conversion, maintainability, licensing, or responsive behavior. A static screenshot cannot reveal keyboard order, focus, assistive-technology output, loading/error states, motion preferences, or code quality.

Mobbin, Page Flows, and Screenlane can inform flow questions, but product screenshots and interaction captures remain third-party copyrighted material. Do not redistribute them in plugin artifacts or use them as pullable templates. Corporate/product/event marketing—including OpenAI Build Week—is accepted only when a user supplies it as a visual case study for that task; it never enters the reusable source catalog.
