# Local review interface

Run `python3 serve.py` and open the printed `/review-app/` URL. The page reads local generated artifacts and never writes stable knowledge.

The interface is deliberately optional. It makes provenance, coverage, and audit evidence easier to inspect; it is not a decorative product dashboard. A developer-mode ChatGPT app can be registered later, but `.app.json` must remain empty until a real app ID exists.
